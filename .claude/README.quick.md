# Astra Terra - Quick Reference

**Type:** Real Estate Platform  
**Framework:** Next.js 14 + TypeScript  
**API:** Pixxi CRM Integration

## Quick Start

```bash
npm install
npm run dev  # http://localhost:3000
```

## Key Directories

- `app/components/` - 51+ React components
- `app/api/` - API routes (properties, contact)
- `app/lib/` - Utilities, actions, Firebase
- `public/` - Static assets

## Main Features

1. Property listings (buy/rent)
2. Interactive Leaflet maps
3. Advanced search & filters
4. Pixxi CRM integration
5. Contact forms + reCAPTCHA
6. Blog + admin panel

## Tech Highlights

- GSAP + Framer Motion animations
- React Leaflet for maps
- Cloudinary for images
- Firebase backend
- Vercel deployment

## Environment

See `.env.local` - requires `PIXXI_TOKEN`
